Array Methods

The Map object holds key/value pairs, like a normal object, but unlike normal objects the Map object remembers the order of key/value pairs. The Map object can hold functions, objects and primitives, whereas ordinary objects can only use a string og a symbol as keys.

Create Map, add key/value pair and get the key value 

        
            // This creates a new Map
            const myMap = new Map();

            // The 'a' key will be set to 1
            myMap.set('a', 1);

            console.log(myMap.get('a'));
            // Logs:
            // 1

Delete values from Map 

        
            myMap.delete('a');

Get number of items in a Map 

        
            const myMap = new Map();

            myMap.set('a', 1);
            myMap.set('b', 2);

            console.log(myMap.size);
            // Logs:
            // 2
