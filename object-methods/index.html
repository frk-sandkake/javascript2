.find() & .findIndex() Array Methode

The .reduce() array method loops through the array. the .filter() object remembers the order of key/value pairs. The Map object can hold functions, objects and primitives, whereas ordinary objects can only use a string og a symbol as keys.

Filter used with if statement 

        
            const numbers = [4, 5, 7, 12, 400, 65, 2, 1];

            const filterNumb = numbers.filter((number) => {
              if (number >= 4) {
                return true;
              }
            });

            console.log(filterNumb);

Filter without if statements 

        
            const numbers = [4, 5, 7, 12, 400, 65, 2, 1];

            const filterNumb = numbers.filter((number) => number >= 4)

            console.log(filterNumb);
